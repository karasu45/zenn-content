## はじめに

セキュリティの勉強、続かなくないですか？

書籍を買っても積読、Udemyの講座は途中で止まる、資格の問題集は3日で飽きる。知識が大事だとわかっていても、「面白くない」という壁は高い。

**Threat Chronicle** は、この問題を「ゲームの力」で解決しようとしているプロダクトです。

セキュリティの脅威を擬人化したキャラクターを集め、タワーディフェンスで活用する。その循環の中心に「学習」を据えることで、**遊んでいたら知識が身についていた**という体験を目指しています。

この記事では、Threat Chronicleの設計思想とピボットの経緯を共有します。

## ピボットの背景

### 最初のアイデア: クイズバトル

初期のThreat Chronicleは「クイズに正解すると敵にダメージを与え、倒すとキャラがドロップする」というシンプルなバトル形式でした。

実装してみて気づいた問題:

- **差別化が弱い**: 「セキュリティ × クイズ」は既存サービスが多い
- **リテンション設計が弱い**: 4レッスン終わったら「次は？」がない
- **キャラの使い道がない**: 集めるだけで活用する場がない

「面白い」にたどり着けていなかった。

### ピボット: 学習 × ガチャ × タワーディフェンス

モバイルゲームのリテンション設計を研究した結果、3つの柱に行き着きました。

```
レッスン学習 → クイズ合格 → XP + ジェム獲得
                                    ↓
               ガチャ（ジェム消費） → キャラ入手
                                    ↓
               タワーディフェンス（キャラ配置で防衛） → 報酬
                                    ↓
                              ↻ 周回ループ
```

- **学習**: 知識を得る場（主軸）
- **ガチャ**: 射幸性でログイン動機を作る
- **タワーディフェンス**: キャラの活用場所 + 学習知識の実践

重要なのは、このループの**どこを切っても学習に戻る**こと。ガチャを引くにはジェムが必要で、ジェムを得るにはレッスンかTDをプレイする必要がある。

## 世界観: サイバーアカデミー

ターゲットは**セキュリティ入門したい社会人（20-30代）**。

資格勉強や業務で知識が必要だけど、真面目な教材は続かない層です。子供向けにしすぎると離れるし、堅すぎると既存教材と変わらない。

そこで「**サイバーアカデミー**」という世界観を設定しました。

| ゲーム用語 | アカデミー用語 | 実際の機能 |
|-----------|-------------|-----------|
| レッスン | 講義 | Markdownコンテンツ読了 |
| クイズ | 試験 | 4択問題 × 5問 |
| TD | 演習（実戦訓練） | タワーディフェンス |
| キャラ | 先輩・仲間 | セキュリティツールの擬人化 |
| ランク | 学位 | Trainee → Junior → Analyst → Senior → Expert → CISO |

「サイバーセキュリティ学部に入学して、アナリストとして成長する」というロールプレイ。ユーザーは新入生で、レッスンを修了するたびにランクが上がっていく。

## UI/UX: モダンライト

初期実装はサイバーパンク風のダークテーマ（`#0a0a14` + ネオングリーン）でした。かっこいいけど、「学習アプリ」としては読みづらくて疲れる。

ターゲットが社会人であることを考えると、**仕事の合間にも使えるクリーンなUI**が必要でした。

## ガチャ設計

### 経済バランス

ガチャの核心は「**無課金でも毎日回せる、課金すると快適になる**」設計です。

| 獲得源 | Free | Pro |
|--------|------|-----|
| デイリーログイン | 5ジェム | 10ジェム |
| レッスン完了 | 10 | 15 |
| TDクリア | 5 | 8 |
| 7日連続ログイン | 20 | 30 |

**日次平均**: Free ≈ 35ジェム / Pro ≈ 54ジェム
**ガチャ1回**: 30ジェム / 10連: 270ジェム

Freeでも毎日1回は引ける。Proなら約5日に1回10連が引ける。この「毎日ログインすればガチャが引ける」がリテンションの要です。

### 排出率と天井

| レアリティ | 排出率 | キャラ例 |
|-----------|--------|---------|
| N（ノーマル） | 60% | パスカ（パスワード管理）、ログミ（ログ監視） |
| R（レア） | 28% | ファイア（ファイアウォール）、ペネトラ（ペンテスト） |
| SR（スーパーレア） | 9% | セイラ（SIEM）、クリプタ（暗号化） |
| SSR（ウルトラレア） | 3% | アテナ（MITRE ATT&CK）、レイス（Red Teamer） |

**天井システム**: SSR天井 = 200連（Free）/ 160連（Pro）。SR天井 = 50連。

10連の10枠目はSR以上が確定。重複したキャラはレベルが自動で1上がる。

### 演出: 盛りめ + スキップ可

ガチャ演出はモバイルゲーム的に盛ります。ただしスキップボタンを常時表示して、社会人の時間を尊重。

- **N**: カード周囲に白い光 → さっとフリップ
- **R**: 青い光の粒子
- **SR**: 紫の光 + 画面フラッシュ + 溜め演出
- **SSR**: 金色の爆発 + 画面全体が光る + 振動エフェクト

## キャラクター設計: セキュリティツールの擬人化

キャラクターはセキュリティの概念やツールを擬人化した「先輩・仲間」です。

### 属性システム（4属性）

| 属性 | 色 | 概念 |
|------|-----|------|
| Shield（防御） | 青 | ファイアウォール、暗号化、防御系 |
| Scan（検知） | 緑 | 検知、監視、分析系 |
| Purge（駆除） | 赤 | 駆除、ペンテスト、攻撃系 |
| Logic（知略） | 紫 | 暗号、AI、戦略系 |

### キャラ例

**パスカ（N / Shield）** — パスワード管理の擬人化
> 「12文字以上、大文字小文字数字記号...覚えられないでしょ？だから私がいるの。」

TD能力: 隣接ユニットの防御力UP

**ペネトラ（R / Purge）** — ペネトレーションテストの擬人化
> 「攻撃者の気持ちがわからなきゃ、守れないでしょ？」

TD能力: クリティカル率2倍

**アテナ（SSR / Logic）** — MITRE ATT&CK フレームワークの擬人化
> 「14のTactics、数百のTechniques...攻撃者の行動はすべて分類済み。」

TD能力: 全敵の弱点を可視化（攻撃力30%UP）+ ボスの能力を1つ無効化

フレーバーテキストで「そのキャラのモチーフが何なのか」を自然に伝える。キャラを好きになる → 元ネタに興味を持つ → 学習につながる、という設計です。

## タワーディフェンス: 知識の実践

### 基本ルール

TDは「学んだ知識を実践する場」として設計しています。

1. **配置フェーズ**: 手持ちキャラから選んでスロットに配置
2. **戦闘フェーズ**: 敵が固定パスを進行、キャラが自動攻撃
3. **クイズインタールード**: ウェーブ間にセキュリティ問題1問出題
   - 正解: 全キャラ攻撃力 +20%（次ウェーブ限定）
   - 不正解: バフなし（ペナルティなし）
4. 繰り返し → 最終ウェーブクリアで報酬

ポイントは**クイズインタールード**。戦闘の合間に学習が挟まることで、「遊びながら知識を使う」体験になります。不正解でもペナルティはなし — 学習の失敗を罰しない設計です。

### 敵 = サイバー脅威

敵キャラもセキュリティの脅威をモチーフにしています。

| 敵 | 特殊能力 | 学べること |
|----|---------|-----------|
| マルウェアスライム | なし（基本敵） | マルウェアの概念 |
| フィッシングドローン | 高速移動 | フィッシングの危険性 |
| ランサムロック | 通過した味方を一時暗号化（行動不能） | ランサムウェアの被害 |
| APTシャドウ | ステルス（検知系キャラのみ攻撃可） | APTの潜伏特性 |
| ワームスプリッター | HP半分で2体に分裂 | ワームの自己増殖 |

敵の能力を理解することが攻略のカギになり、それがそのままセキュリティ知識の理解につながります。

## 学習コンテンツ

### 24レッスン（6ステージ × 4レッスン）

| Stage | テーマ | Free/Pro |
|-------|--------|----------|
| 0 | サイバー脅威の世界へようこそ | **Free** |
| 1 | マルウェア大全 | Pro |
| 2 | 有名サイバー攻撃事件簿 | Pro |
| 3 | Web攻撃手法の解剖学 | Pro |
| 4 | ネットワーク攻撃と暗号の基礎 | Pro |
| 5 | 防御の哲学 | Pro |

Stage 0は完全無料。4レッスンでサイバー脅威の全体像をつかめる。

Pro（月額980円）にすると、WannaCryやSolarWindsなどの実際の事件、SQLインジェクションやXSSの仕組み、ゼロトラストの考え方まで学べる。各レッスン5問のクイズ付きで合計120問。

### コンテンツ方針

Threat Chronicleのコンテンツは**エンタメ寄り**です。

- 脅威の知識・攻撃手法・歴史・事件簿
- 「怖い」「すごい」「なるほど」の感情を動かす
- 一本道方式（キャリアパス分岐なし）

設計ポリシーは「**Freeでも毎日ガチャが引ける。Proにすると全部快適になる**」。

コンテンツで壁を作りつつ、ゲーム要素は制限を緩めに設定。「これ面白いな、全レッスンやりたいな」で課金してもらうのが理想。

## おわりに

Threat Chronicleは「セキュリティ学習を続けられない」問題を、ゲーム設計の力で解決しようとしています。

- 学ぶ → ジェムが貯まる → ガチャが引ける → キャラが増える → TDが楽しくなる → もっと学びたくなる

この循環ループが回り始めたとき、「勉強」は「遊び」に変わる。
