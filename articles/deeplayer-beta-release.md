## はじめに

Claude Codeが登場してから、AIを活用した開発スタイルは大きく変わりつつあります。ターミナルからAIに指示を出し、ファイルの作成・編集・テスト実行・Git操作までをエージェントに任せる。この新しいワークフローの可能性に多くのエンジニアが気づき始めています。

しかし、いざClaude Codeを使い始めようとすると、こんな壁にぶつかりませんか？

- **公式ドキュメントが英語**で、体系的に学ぶのが難しい
- CLAUDE.md、hooks、MCPなど**設定ファイルの全体像**がつかめない
- 「とりあえず触ってみた」の先にある**実践的なベストプラクティス**がわからない
- 手元に環境を用意するのが面倒で、**気軽に試せない**

こうした課題を解決するために、**DeepLayer** というインタラクティブ学習プラットフォームを作りました。

https://deeplayer.jp

## DeepLayerとは

DeepLayerは、Claude Codeの使い方を**日本語で、ブラウザ上で、体系的に学べる**プラットフォームです。Progateのような体験学習スタイルで、スライドで知識をインプットしたあと、ブラウザ内のリプレイターミナルですぐにアウトプットを体験できます。

### 3つの特徴

**1. 環境構築ゼロ、ブラウザだけで学べる**

Claude Codeのローカルインストールは不要です。ブラウザ内にリプレイターミナルを用意しており、実際のClaude Codeの操作フローをボタンひとつで体験できます。コマンドの入力から出力の確認まで、ステップバイステップで進められます。

**2. 「Layer」構造の段階的カリキュラム**

コンテンツは3つのLayerに分かれています。

| Layer | テーマ | 内容 |
|-------|--------|------|
| Layer 1：基礎 | Claude Codeの基本 | AI駆動開発とは何か、基本操作、コンテキスト管理、指示の出し方、タスク分解 |
| Layer 2：実践 | 設定とワークフロー | CLAUDE.md設計パターン、プロンプトエンジニアリング、hooks活用、Git連携、チーム開発 |
| Layer 3：応用 | 高度な活用 | MCP、マルチエージェント、カスタムエージェント構築（Coming Soon） |

「深く潜るほど、高度なスキルが身につく」というコンセプトで、Layer 1の基礎から順に進められます。

**3. すべて日本語**

スライド、ターミナルの説明、ヒントまで、すべて日本語です。英語の公式ドキュメントを読み解く必要はありません。

## 技術スタックの紹介

個人開発者の方の参考になるかもしれないので、技術構成を簡単に紹介します。

### フロントエンド・バックエンド

- **Next.js 16**（App Router）+ **TypeScript** + **Tailwind CSS 4**
- React 19のServer Componentsをデフォルトにし、Client Componentsは最小限に抑えています
- ダークテーマ対応（next-themes）

### リプレイターミナル

ここが一番こだわったポイントです。

当初はClaude Haiku APIを使ったリアルタイムLLMシミュレータを実装していましたが、コスト・安定性・UXの観点からリプレイ方式に刷新しました。

```
レッスンページ読み込み
  ↓
lesson_steps テーブルから
ステップデータ取得
（command, output, description）
  ↓
リプレイターミナルに表示
  ↓
ユーザーが「実行」ボタンをクリック
  ↓
出力を1行ずつアニメーション表示
  ↓
全ステップ完了 → 進捗記録
```

各レッスンの「やってみよう」パートでは、「悪い指示 vs 良い指示」「タスク分解の4ステップ」「CLAUDE.md作成の流れ」など、実践的な操作フローを録画済みステップとして用意しています。

APIコールが不要なので、**コストゼロ・無制限に何度でも復習可能**です。

- **Stripe**（Checkout + Webhook + Promotion Code）でフリーミアム課金

## 使い方の流れ

アカウント登録後、コース一覧から「Claude Code マスターコース」を選択してLayer 1から学習開始。左側のスライドで知識をインプットし、一部のレッスンではリプレイターミナルで実際の操作フローを体験できます。完了すると修了証が発行され、Xでシェアもできます。4レッスンまでは無料です。

## なぜ作ったのか

私自身がClaude Codeのヘビーユーザーで、日々の開発で多用しています。hooks、CLAUDE.md、MCPなどを駆使して生産性を高めていますが、これらのノウハウを体系的にまとめた日本語リソースがほとんど存在しないことに気づきました。

Claude Codeは非常に強力なツールですが、「ただ質問に答えてもらうだけ」の使い方で止まっている人が多いように感じます。CLAUDE.mdでプロジェクトの規約をエージェントに伝える、hooksでコード品質を自動チェックする、MCPで外部サービスと連携する ── こうしたテクニックを知っているかどうかで、開発効率は大きく変わります。

DeepLayerが、Claude Codeを「なんとなく使っている」状態から「戦略的に活用できる」状態への橋渡しになればうれしいです。
